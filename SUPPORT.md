# Support

Need help with HamTab? Here's how to get support.

## Community

- **Discord** — Join us at [discord.gg/GcX9cHtT](https://discord.gg/GcX9cHtT) for real-time help and discussion
- **Wiki** — Check the [project wiki](https://github.com/stevencheist/HamTabv1/wiki) for guides and documentation
- **User Guide** — Download the PDF user guide from the app's Help menu

## Bug Reports

Found a bug? [Open an issue](https://github.com/stevencheist/HamTabv1/issues/new?template=bug_report.yml) using the bug report template.

## Feature Requests

Have an idea? [Open a feature request](https://github.com/stevencheist/HamTabv1/issues/new?template=feature_request.yml).

## Security Issues

For security vulnerabilities, please see [SECURITY.md](SECURITY.md) for responsible disclosure instructions. Do not open a public issue for security problems.
